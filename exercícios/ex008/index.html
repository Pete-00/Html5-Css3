Principais formatações

negrito / destaque

nesta frase podemos observar o texto em negrito com a tag <b>(não semântica) só tem forma e não um significado
já nesta outra frase temos o texto em destaque com a tag <strong>(semântica) com significado

Itálico / Ênfase

Nesta frase podemos ver um termo em italico usando a tag <i> (nao semântica)
nesta frase temos um termo em Ênfase com a tag <em>

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>, e para mudar a cor do marcador do texto usando style logo depois de MARK

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo com a tag <big>(obs.:a tag big já está obsoleta portanto evite usa-la) e a tag <small>

Texto Deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins>

também existe a tag U(não semântica e diferente de ins)

Texto sobrescrito

Para inserir coisas do tipo como contas matemáticas: x20+3 com a tag <sup>

Texto subscrito

para inserir coisas do tipo como elementos químicos: H20 com a tag <sub>

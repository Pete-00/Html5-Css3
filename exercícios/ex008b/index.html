Citações simples

Com a tag <q> podemos colocar aspas nos textos (colocamos a tag ao inves de colocar as aspas manualmente para dar sentido ao código e não só forma como colocando as aspas manualmente) Ex: Lorem ipsum dolor sit, amet consectetur adipisicing

Citações completas

Com a tag <blockquote> podemos deixar claro que é uma citação para um determinado autor

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Reiciendis praesentium unde ex corrupti necessitatibus, ullam culpa maiores nesciunt facilis vel, sed tempora laudantium eveniet, nisi ea ad. Quibusdam, velit totam?

Abreviações

com a tag <abbr> podemos dar significado para siglas, quando o visitante do site passar o mouse em cima da sigla a tag <abbr> vai mostrar o significado.

Ex: Estou cursando HTML e CSS! Estou adorando.

Texto invertido

com a tag <bdo>podemos inverter um texto dependendo da direção ltr(left to right)e a rtl(right to left)

Ex: Estou amando viver(rtl) Estou amando viver(ltr)

Listas

Listas Ordenadas

para criarmos lista ordenadas usamos a tag <ol> para delimitarmos a lista e <li> para identificar cada item da lista. Ex:

  1. Acordar
  2. ligar pro joao
  3. tomar banho
  4. escovar os dentes
  5. tomar café

com o parametro type podemos mudar a formatação. Ex:

  1. Acordar
  2. tomar banho
  3. escovar os dentes
  4. tomar café

e com o paramentro start podemos decidir onde nossa lista começa. Ex:

  1. Acordar
  2. tomar banho
  3. escovar os dentes
  4. tomar café

Listas não ordenadas

para criarmos listas não ordenada(onde a ordem não importa) podemos usar da tag <ul> para delimitarmos a lista e <li> para definirmos os itens da lista

  • site
  • html
  • css
  • javascript

caso queira podemos personalizar o ponto de cada item da lista usando o parametro type disc, circle ou square. Ex:

  • site
  • html
  • css
  • javascript

Segunda parte da aula, listas mistas / listas de definição

listas mistas

para fazermos uma lista dentro de um item de outra lista basta abrirmos a tag <ol> ou a tag <ul> dentro da lista. Ex:

    <ol>
  1. segunda
      2. <ol>
    1. matemática
    2. culturas
    3. cdpef
      4. </ol>
  2. terça
      3. <ul>
    • PPIP
    • matemática
    • psicologia
      • </ul>
  3. quarta
  4. quinta
    5. </ol>

Lista de definição

na lista de definição temos 3 tags. A primeira delas é a própria lista denominada pela tag <dl> (dl definition list), a segunda sendo os termos (ou itens da lista) denominados pela tag <dt>(definition term) e por ultimo as descrições dos termos denominados pela tag <dd> (definition description).
